# 舔狗的艺术
学习vue-cli的成果 🥝

vue / vue-router / elementUI

预览地址：https://dogart.me

## 使用方式：

安装vue-cli （见官网安装方式）

拉取项目到本地

用vue-cli 导入项目

执行 npm update 更新依赖

使用 vue ui 预览、编译项目

编译后在dist目录里，可以导入其他项目二次开发实现前后端分离的项目

## 不想修改和编译

你可以点击 [这里](https://github.com/coolseamonster/dogartvue/releases/download/zip/dist.zip) 下载编译好的文件（不推荐）
